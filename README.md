# Fantasy football holographic cards

Based heavily on https://github.com/simeydotme/pokemon-cards-css but converted to React with some performance optimizations

### A collection of advanced CSS styles, applied with SvelteJS.
Uses CSS Transforms, Gradients, Blend-modes and Filters to simulate the various Holofoil effects found
in pokemon cards
